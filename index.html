জাভাস্ক্রিপ্ট একটি ক্লায়েন্ট সাইড স্ক্রিপ্টিং ল্যাঙ্গুয়েজ। বর্তমানে এটি যে কোন অপারেটিং সিস্টেমে রান করে এবং অন্যান্য হাই লেভেল প্রোগ্রামিং ল্যাঙ্গুয়েজের মত সমস্ত বৈশিষ্ট্য এতে বিদ্যমান। bortomane javascript er onk cahida.

First Step: JavaScript code ব্রাউজারে রান করলে প্রথমে JavaScript Engine এর মধ্যে parser দ্বারা কোডকে রিসিভ করে, সবগুলো কোড পড়ে। যেহেতু জেএস একটি ইন্টারপ্রেটেড ল্যাঙ্গুয়েজ, তাই যখনই কোন ভুল পায় সাথে Error throw করে বাকী কোডকে চলতে দেয় না। ভুল না হলে পরবর্তী ধাপে চলে যায়। Second Step: parser দ্বারা কোন ভুল ধরা না পড়লে রিসিভ করা কোডগুলোকে Abstract Syntax Tree (AST) তে পরিণত করে। Third Step: অতপর AST থেকে Machine Code এ কনভার্ট করে। তারপর ব্রাউজারে আউটপুট দেখায়।

Local Storage ডাটা স্টোর করে rakte পারে 10mb কিন্তু Session Storage স্টোর করেতে পারে 5mb। Local Storage এর ডাটা স্থায়ী থাকে, কিন্তু Session Storage এর ডাটা স্থায়ী থাকে না। Local Storage এর ডাটা ম্যানুয়ালি বা JavaScript দিয়ে delete করতে হয়। কিন্তু Session Storage এর ডাটা ব্রাউজারের ট্যাব বন্ধ করলেই হাওয়া হয়ে যাবে।
